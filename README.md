## Run / Installation
How to run ?

```bash
> git clone https://github.com/tegal1337/Youtube-Auto-Commenter/
> cd Youtube-Auto-Commenter
> npm install
> node main.js
```

dont forget to change the google username and password in ` config.js`

```javascript
module.exports = {
  keywords: "keywords",
  usernamegoogle: "username",
  passwordgoogle: "pass",
};

```
